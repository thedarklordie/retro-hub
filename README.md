# 🕹 Retro Hub

Welcome to **Retro Hub**. a collection of retro-style websites, scratch projects, and maybe more

## 🌐 What is this?

Retro Hub is a simple directory site inspired by early internet forums and old web pages (2000-2024 and earlier).

It is designed to feel:
- simple
- nostalgic
- lightweight

## 🎯 Goal

To recreate the feeling of early internet communities where websites were simple, fun, and personal.
